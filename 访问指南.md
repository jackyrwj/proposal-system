## 📂 项目文件

- `app/` - 页面组件
- `components/` - 共用组件（Header、Footer）
- `types/` - TypeScript类型定义
- `lib/` - 工具函数
- `public/` - 静态资源（图片等）

## ⚠️ 注意事项

1. **图片资源**: 需要从原系统复制图片到 `public/` 目录
   - `title.png` - 网站标题图片
   - `news.jpg` - 新闻默认图片

2. **Mock数据**: 当前使用模拟数据，需要连接后端API

3. **端口冲突**: 如果3000端口被占用，可以修改：
   ```bash
   npm run dev -- -p 3001
   ```

## 🐛 故障排除

### 如果页面无法加载

1. **检查服务器是否运行**
   ```bash
   netstat -ano | findstr :3000
   ```

2. **重启服务器**
   - 按 `Ctrl + C` 停止
   - 重新运行 `npm run dev`

3. **清除缓存**
   ```bash
   rm -rf .next
   npm run dev
   ```

4. **检查防火墙设置**
   - 确保端口3000未被阻止
